-------About the Project-------

The Shopping Cart project is a front-end web application that simulates the basic working of an online shopping system.  
Users can browse products, add them to the cart, update quantities, and view the total price in real time.  
This project is ideal for beginners to understand **DOM manipulation, JavaScript logic, and UI interaction**.

--------Features-----------

- Display products dynamically
- Add products to the shopping cart
- Increase or decrease product quantity
- Remove items from the cart
- Automatic price calculation
- Responsive and user-friendly UI
- Clean and structured code

------Technologies Used----------

- HTML – Structure of the web pages  
- CSS – Styling and layout  
- JavaScript – Application logic and DOM manipulation  

---------How It Works----------

1. Products are loaded from the data files.
2. Products are displayed on the homepage.
3. When the user clicks Add to Cart, the item is added to the cart.
4. Cart updates automatically when quantity changes.
5. Total price is calculated in real time using JavaScript.
